# webpwn3r
WebPwn3r - Web Applications Security Scanner.
</br>[@bugyellow](https://twitter.com/bughuang1)

Usage example:
```
~/go/bin/webpwn3r URL
cat URL.txt | ~/go/bin/webpwn3 
```
install
```
go get github.com/bugyellow/webpwn3r
```
